Welcome to my React application workspace! This repository is a comprehensive monorepo housing 20 distinct web applications. 
It ranges from essential utility calculators and productivity tools to data-driven business trackers and interactive forms.
This project showcases clean React state management, dynamic UI rendering, form validation, and complex mathematical algorithms.
📱 App Directory & DescriptionsHere is the complete breakdown of every application built into this workspace:🎛️ Central HubDashboard: The central launchpad of the workspace. 
It features quick-access navigation cards for all 20 apps, dynamic search filtering, and light/dark mode toggling.


🧮 Calculators & ConvertersCalculator: A fully-featured arithmetic calculator supporting basic operations ($+$, $-$, $\times$, $\div$), decimal inputs, and operation chaining.BMI Calculator: 
A health utility that calculates Body Mass Index based on height and weight inputs, dynamically displaying weight classification (Underweight, Normal, Overweight, Obese) using visual color zones.
Age Calculator: A precision tool that calculates exact age down to the years, months, and days based on a selected date of birth.Temperature Converter: 
A real-time bidirectional converter supporting Celsius, Fahrenheit, and Kelvin scales.
Currency Converter: An application that converts values between global currencies using live API exchange rates or custom-defined rates.
Unit Converter: A multi-category conversion tool supporting length (meters, miles, feet), weight (kg, lbs, oz), and volume.
💼 Finance & Utility CalculatorsSalary Calculator: A payroll calculation tool that estimates take-home pay by factoring in gross monthly/hourly salary, tax deductions, and insurance contributions.
Loan Calculator (EMI): An amortization calculator that computes monthly loan payments (Equated Monthly Installments) based on principal amount, interest rate, and tenure.
Electricity Bill Estimator: A utility calculator that determines monthly energy bills by multiplying kilowatt-hour (kWh) consumption against tiered pricing slabs.
Water Bill Estimator: A municipal utility tool that estimates water bills based on total consumption (in gallons or liters) and local base service charges.
Productivity & ManagementToDo App: A classic task manager featuring task creation, editing, status toggling (complete/incomplete), local storage persistence, and filter views.
Attendance Tracker: A digital register for teachers or managers to log daily student/employee attendance with present/absent stats.Expense Tracker: 
A personal finance manager that logs income and expenses, categorizes transactions, and displays a real-time net balance.Phone Book: 
A contact directory with add, edit, delete, and search capabilities.🎮 Interactive & GamifiedVoting System: 
A real-time poll app where users can vote on different options, displaying live progress bars and percentage calculations.
Quiz App: A multiple-choice trivia challenge with a timer, real-time score tracking, and a detailed results summary.Counter: 
A simple yet crucial React state showcase featuring increment, decrement, and reset functions.🛒 E-Commerce & GradingProduct Catalog: 
A dynamic showcase of products featuring search filters, category sorting, and an interactive "Add to Cart" state.Grade Calculator: 
An academic tool that averages homework, exam, and quiz scores to output final percentages and letter grades (A, B, C, etc.).


🛠️ Key Technical Features DemonstratedState Management: 
Heavy use of React's useState, useEffect, and custom hooks.
Responsive UI: Designed to work flawlessly on both mobile screens and wide desktop monitors.
Routing & Hub Navigation: Seamless single-page application (SPA) experience to jump between tools.
Data Persistence: Many tools leverage localStorage so user data (like to-dos and expenses) stays saved even after a page refresh.🚀
